Describing the compilation process :
* 0-preprocessor runs a C file through the preprocessor and save the result into another file. Thereafter saving it in a .out file using the option -E
* 1-compiler compiles a C file but does not link. Thereafter saving the file in .o file using the option -c.
* 2-assembler generates the assembly code of a C code and save it in an output file. Thereafter, saving it in .s file using -S flag.
